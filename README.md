# AAVE Ebon

Don't we grow up too fast? One minute you're cool — next minute the comments section's speaking a whole different language and you're just vibing there, lost. **AAVE Ebon** is an AI-powered AAVE dictionary that actually respects the culture: search any slang, get a real definition with context and origins, and if it's not in the database yet, the AI explains it and saves it so the next person doesn't have to ask. Sign in and it gets personal — track your streak, build a word bank, quiz yourself, earn badges, and actually *learn* instead of just tab-closing in confusion. Urban Dictionary meets Duolingo, built different.
